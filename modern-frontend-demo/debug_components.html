// App.tsx - 物性推理页面结构 case 'reasoning': return ( <div className="space-y-6"> <PhysicalReasoningAlgorithmExplanation /> // 新增的算法讲解 <StepByStepPhysicalReasoning /> // 原有的分步演示 </div> )

🚨 常见问题和解决方案

问题1：组件不显示

原因：浏览器缓存了旧版本的代码

解决：

  1. 硬刷新：Ctrl/Cmd + Shift + R
  2. 开发者工具 → Network → 勾选 "Disable cache"
  3. 清除 localStorage: localStorage.clear()

问题2：热更新失效

原因：Vite的HMR（热模块替换）缓存问题

解决：

  1. 停止开发服务器（Ctrl+C）
  2. 删除缓存：rm -rf node_modules/.vite
  3. 重新启动：npm run dev

问题3：组件导入错误

检查：确保文件路径正确

import PhysicalReasoningAlgorithmExplanation from '@/components/features/PhysicalReasoningAlgorithmExplanation'

✅ 验证步骤

  1. Step 1: 确认文件存在

    ls -la src/components/features/Physical*.tsx

    应该看到：

    • PhysicalReasoningAlgorithmExplanation.tsx
    • PhysicalPropertyVisualization.tsx
    • PhysicalReasoningDemo.tsx（如果存在）

  2. Step 2: 清理并重启

    # 终端1
Ctrl+C （停止服务器）
rm -rf node_modules/.vite
rm -rf dist
npm run dev

# 终端2（如果API服务器也需要重启）
Ctrl+C
python demo_frontend.py

  3. Step 3: 浏览器操作

    1. 打开新的隐私窗口/无痕模式
    2. 访问 http://localhost:3000
    3. 直接点击"🧩 物性推理"标签
    4. 查看是否显示两个组件

🔍 开发者工具检查

在浏览器中按 F12，检查以下内容：

  1. Console：是否有红色错误信息
  2. Network：是否所有JS文件都加载成功（状态码200）
  3. Elements：在React DevTools中查看组件树
