🧩 基于物性关系的推理算法

通过物性关系连接已知和未知，实现高效可解释的数学推理

算法核心思想

通过引入物性关系，在断开的已知和未知之间建立连接通路， 然后沿着找到的路径进行推理，忽略无关的推理分支。

算法公式

题目 → Phrase + Entity → 直陈关系 → +物性Entity → 物性Graph

↓

物性推理扩展 → 找通路(已知→未知) → 沿路径推理 → 答案

算法五步骤

1
题目分解
提取phrase和entity，获得直陈关系
2
构建物性Graph
直陈关系 + 物性entity
3
物性关系推理
通过物性规律扩展Graph
4
寻找通路
找到已知到未知的路径
5
精简推理
只保留路径，忽略无关推理

三条核心推理规则

规则1：守恒扩展
什么时候用：看到"一共"、"总共"、"全部"等关键词
怎么用：建立部分与整体的守恒关系
示例应用：5个苹果 + 3个苹果 = 总数（守恒定律）
规则2：因果链接
什么时候用：存在动作或变化时
怎么用：建立原因→结果的因果关系
示例应用：数量×单价 → 因果关系 → 总价
规则3：时空连续
什么时候用：涉及时间或空间变化时
怎么用：建立连续性关系
示例应用：100升 ÷ 20升/时 → 时间关系 → 5小时

算法应用示例

示例1：苹果数量问题
题目：小明有5个苹果，妈妈又给了他3个苹果，现在小明一共有几个苹果？

应用规则：规则1（守恒扩展）- 识别"一共"关键词

推理过程：
  1. 提取实体：小明、苹果、5个、3个
  2. 识别"一共"→ 应用守恒扩展规则
  3. 建立关系：5个 + 3个 → 守恒 → 总数
  4. 沿路径推理：5 + 3 = 8
  5. 忽略无关信息（小明、妈妈等）
答案：8个苹果
示例2：水池排水问题
题目：水池有水100升，每小时流出20升，几小时后水池为空？

应用规则：规则3（时空连续）- 涉及时间变化

推理过程：
  1. 提取实体：水池、100升、20升/时
  2. 识别时间变化 → 应用时空连续规则
  3. 建立关系：总量 ÷ 速率 → 时间关系 → 时间
  4. 沿路径推理：100 ÷ 20 = 5
  5. 忽略无关信息（水池等）
答案：5小时
示例3：购物问题
题目：小红买了3支笔，每支笔5元，她一共花了多少钱？

应用规则：规则2（因果链接）- 数量和单价决定总价

推理过程：
  1. 提取实体：小红、笔、3支、5元/支
  2. 识别因果关系 → 应用因果链接规则
  3. 建立关系：数量×单价 → 因果 → 总价
  4. 沿路径推理：3 × 5 = 15
  5. 忽略无关信息（小红、买等）
答案：15元
