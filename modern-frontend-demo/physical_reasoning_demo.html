🧩 基于物性关系的推理算法

算法流程

1
题目解析 → 分解出 phrase 和 entity → 识别直陈关系
2
物性增强 → 加入物性关系（守恒、因果、时空等）→ 构建物性 Graph
3
关系推理 → 通过物性关系扩展 → 找到隐含的连接
4
路径发现 → 寻找已知到未知的通路 → 确定求解路径
5
推理执行 → 沿路径推理 → 忽略无关推理 → 得出答案

📋 示例演示

例题 1：苹果数量问题

题目：小明有5个苹果，妈妈又给了他3个苹果，现在小明一共有几个苹果？

初始苹果(5个) → 加法运算 → 总苹果数(?)
购买苹果(3个) ↗
物性关系：守恒定律（部分之和等于整体）

推理路径：初始苹果 → [守恒] → 总苹果数 ← [守恒] ← 购买苹果

答案：5 + 3 = 8个苹果

例题 2：水池排水问题

题目：水池有水100升，每小时流出20升，几小时后水池为空？

初始水量(100升) → 流出速率(20升/时) → 时间(?小时)
物性关系：时间流逝（线性变化）

推理路径：初始水量 → [时间关系] → 流出速率 → [因果] → 时间

答案：100 ÷ 20 = 5小时

例题 3：购物价格问题

题目：小红买了3支笔，每支笔5元，她一共花了多少钱？

数量(3支) → 乘法运算 → 总价(?元)
单价(5元/支) ↗
物性关系：价值守恒（价值交换）

推理路径：数量 → [因果] → 总价 ← [因果] ← 单价

答案：3 × 5 = 15元

💡 核心洞察

物性关系的力量：通过引入物理世界的基本规律（守恒、因果、时空连续性等）， 我们能够在看似断开的已知和未知之间建立桥梁。这种方法不仅模拟了人类的直觉推理， 还提供了可解释的推理路径。

时间和空间的连续性

  • 运动轨迹
